About Smart Road System

Project Overview

This project focuses on collecting rainwater from roads and reusing it for cleaning and irrigation. It helps reduce water waste and improves city maintenance.

System Design

The system includes drainage channels, underground storage, and a distribution system connected to cleaning and irrigation units.

Software Contribution

The web-based dashboard simulates system monitoring and control, allowing users to visualize water levels and system operations.
