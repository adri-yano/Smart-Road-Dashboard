# Smart Road System with Rainwater Storage for Automatic Cleaning and Plant Irrigation

## Integrated Engineering Team Project

This project is a multidisciplinary engineering system that combines Civil Engineering and Software Engineering concepts to create a smart and sustainable road infrastructure solution.

The system collects rainwater from road drainage channels, stores it in underground reservoirs, and reuses the stored water for:

* Automatic road cleaning
* Irrigation of roadside plants

A web-based dashboard simulates the monitoring and control system of the project.

---

# Project Objective

The main objective of this project is to improve water utilization and urban road maintenance through automation and smart infrastructure.

# How the Simulation Works

* Rainfall increases the water level in the storage tank.
* Cleaning and irrigation operations consume stored water.
* The system automatically stops operations when water becomes low.
* The dashboard updates system status in real time using JavaScript logic.
